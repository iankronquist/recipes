    About this site


    After keeping my recipes on github for a few months, I decided it was time to start a food blog! I was also scheduled to give a talk on jekyll at our local Linux Users group a few days later, and figured I should actually build a site with the tool before talking about it. This site is purely static -- everything from the search bar to the contact form to the blog itself is static! It's even hosted on Github Pages, so I really can't use any server-side logic. I'm hoping to post more about how I built the site soon, but in the meantime you can check the site out on github, and contact me using the shiny contact form with any questions!
